| HTTP Method | Endpoint  | Description                                                                                                                                | Request Body                                                                                                           | Success Response          | Error Responses                                                                   |
|-------------|-----------|--------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|---------------------------|-----------------------------------------------------------------------------------|
| `POST`      | `/api/v1/place`  | Places the robot at the given coordinates and facing direction.                                                                            | raw JSON: `{ "x": <int>, "y": <int>, "f": "NORTH"\|"EAST"\|"SOUTH"\|"WEST" }` Example: `{"x":1,"y":2,"f":"EAST"}` | `200 OK` (empty body). Note: even if coordinates are outside the tabletop the endpoint returns 200; the robot will simply not be placed.     | `400 Bad Request` – missing/empty request body, null `f`, or invalid direction string. |
| `POST`      | `/api/v1/move`   | Moves the robot one unit forward in its current direction (ignored if robot not placed or move would fall).                                | none                                                                                                                   | `200 OK` (empty body)     | –                                                                                 |
| `POST`      | `/api/v1/left`   | Rotates the robot 90° counter‑clockwise.                                                                                                   | none                                                                                                                   | `200 OK` (empty body)     | –                                                                                 |
| `POST`      | `/api/v1/right`  | Rotates the robot 90° clockwise.                                                                                                           | none                                                                                                                   | `200 OK` (empty body)     | –                                                                                 |
| `POST`      | `/api/v1/reset`  | Removes the robot from the tabletop, i.e. unplaces it.                                                                                     | none                                                                                                                   | `200 OK` (empty body)     | –                                                                                 |
| `GET`       | `/api/v1/report` | Returns the current position and facing direction as a string (e.g. `"2,3,NORTH"`). When the robot is not placed returns `"ROBOT MISSING"`. | none                                                                                                                   | `200 OK` with string body | –                                                                                 |
